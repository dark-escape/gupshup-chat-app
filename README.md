# GUPSHUP  
Android chat app with custom server and firebase as a push service.
## Features:
<ul>
<li>Private Messaging.</li>
<li>Scheduling Messages.</li>
<li>Start a chat from contacts.</li>
<li>Start a chat by directly entering number.</li>
</ul>
